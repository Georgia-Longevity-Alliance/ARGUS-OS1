# ARGUS V8 — Light-Sheet на базе OpenFlexure

**Концепт.** 2026-07-21.

---

## Принцип

```
Лазер 488 nm → цилиндрическая линза → световой лист (~3 мкм)
→ освещает 1 плоскость эмбриона
→ камера перпендикулярно листу
→ образец движется по Z
= 3D стек, минимальная фототоксичность
```

## Отличия от LLSM

| | LLSM (Betzig) | ARGUS V8 |
|---|---|---|
| Решётка | Spatial light modulator ($50K) | Нет |
| Лист | Lattice pattern | Gaussian (цилиндрическая линза) |
| Толщина листа | ~1 µm | ~3-5 µm |
| Стоимость | $500K-1M | **~$3K дополнительно** |
| Для эмбриона C. elegans | Избыточно | **Достаточно** |

## Преимущества

- Световой лист → фототоксичность ×10 ниже чем wide-field
- 3D time-lapse без photobleaching
- Идеально для эмбриона C. elegans
- Открытая архитектура (OpenSPIM-совместимо)
- Трекинг центриолей на уровне LLSM-данных
