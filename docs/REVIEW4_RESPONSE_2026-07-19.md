## ПОСТАТЕЙНЫЙ ОТВЕТ НА ЗАМЕЧАНИЯ

### Замечание 1: 3.1% SAI «цифра из ниоткуда»

**Ответ:** Частично СОГЛАСНЫ. Уточнили.

Thomas & Meraldi 2024 (PMID 39012627) сообщают:
- SAI = 2.7 ± 5.5% (n=28, P=0.0145, early condition)
- SAI = 3.8 ± 5.6% (n=28, P=0.0012, t=-1 min before anaphase)

Взвешенное среднее ≈3.1%. Именно это значение использовалось как сокращение.

**Исправление (v57):**
- Central Hypothesis: «SAI = 2.7 ± 5.5% (early metaphase) to 3.8 ± 5.6% (late anaphase), weighted mean ≈3.1%»
- Pathway A: явно указаны оба значения с n и P
- Pilot 3: прямое измерение SAI в ARGUS-LP_OS для верификации воспроизводимости

### Замечание 2: Fine-Gray + frailtypack

**Ответ:** СОГЛАСНЫ. Исправили.

`frailtypack` (CRAN) действительно НЕ поддерживает Fine-Gray с фрайлти для кластеризованных данных в стабильной версии. Это экспериментальная функция.

**Исправление (v57):**
- **Primary:** Cox PH + cluster-robust standard errors (`coxph(... + cluster(IslandID))`)
- **Sensitivity:** Fine-Gray без фрайлти (`cmprsk`) + bootstrap для кластеризации
- **Убрано:** frailtypack из первичного анализа
- **Ссылка на Rondeau 2012:** оставлена только как reference

### Замечание 3: Адаптивный дизайн (Cui-Hung-Wang)

**Ответ:** СОГЛАСНЫ. Упростили.

**Исправление (v57):**
- Фиксированный N=300
- Промежуточный анализ: ТОЛЬКО досрочная остановка (O'Brien-Fleming)
- Остановка по эффективности: α=0.005 при N=150
- Остановка по бесполезности: conditional power <20%
- **БЕЗ адаптивного увеличения N** (Cui-Hung-Wang удалён)
