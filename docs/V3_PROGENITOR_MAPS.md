# V3.0 — ПРОГЕНИТОРНЫЕ КАРТЫ

**Date:** 2026-07-20
**Задача:** Создание карт прогениторных клеток — проследить судьбу каждой дочерней клетки после асимметричного деления.
**Status:** Принятие решения по методу.

---

## Три подхода

| # | Подход | Суть | Какие вопросы отвечает |
|---|--------|------|----------------------|
| **1** | Абляция | Убиваем одну дочернюю клетку лазером → смотрим что стало с популяцией | Какая дочь критична для прогениторного пула? |
| **2** | Разделение | Фемтосекундный лазер + манипуляторы → разделяем сестринские клетки → отслеживаем каждую | Какова судьба каждой дочери? Полное дерево. |
| **3** | мтДНК-метка | Трансплантация мутантных митохондрий → дочерние клетки получают разную гетероплазмию → секвенирование | Lineage tracing без вмешательства. |

---

## Анализ

### Подход 1: Абляция дочерних клеток

```
Делится клетка → две дочери (A и B)
                ↓
         Убиваем A лазером
                ↓
         Смотрим: B делится? Дифференцируется? Умирает?
```

| Плюсы | Минусы |
|--------|--------|
| Технически проще всех | Необратимо — теряем информацию о судьбе убитой клетки |
| Фемтосекундный лазер есть в v2.0 (добавить) | Ответ: «клетка А была НЕОБХОДИМА». Не «какова её судьба». |
| Дешёво: $5-15K за лазерный модуль | Не строит полное прогениторное дерево |
| Быстрый ответ: 2-3 недели на эксперимент | Артефакт: убийство клетки ≠ её естественная судьба |

**Вердикт:** годится для hypothesis testing ПОСЛЕ того как карта построена. Не для построения самой карты.

---

### Подход 2: Разделение сестринских клеток

```
Делится клетка → две дочери (A и B) на микроостровке
                ↓
      Фемтосекундный лазер режет межклеточный контакт
                ↓
      Манипулятор растаскивает A и B в разные лунки
                ↓
      Отслеживаем каждую 48-72h: деление, дифференцировка, смерть
```

| Плюсы | Минусы |
|--------|--------|
| Сохраняем ОБЕ клетки живыми | Сложно технически |
| Полное прогениторное дерево | Фемтосекундный лазер: $15-30K |
| Судьба каждой дочери известна | Манипуляторы: 3-осевые + микрокапилляры + пневматика: $5-10K |
| Можно повторить рекурсивно (3-4 поколения) | Каждая операция — минуты. 100 клеток = часы работы |
| **Золотой стандарт:** единственный метод дающий ПОЛНУЮ карту | Лазерные пинцеты (оптические) могут заменить механические манипуляторы — +$20-30K |

**Вердикт:** **ОСНОВНОЙ МЕТОД для v3.0.** Единственный способ построить полную прогениторную карту.

**Аналоги:**
- LUMICKS C-Trap (~$200K) — оптические пинцеты + флуоресценция. Может удерживать и перемещать клетки.
- ZEISS PALM MicroBeam — лазерная микродиссекция + манипулятор.
- **ARGUS v3.0 = open-source аналог за $30-50K.**

---

### Подход 3: мтДНК-метка

```
Клетка с мутантными митохондриями (гетероплазмия 50%)
                ↓
        Деление → случайное распределение мтДНК
                ↓
   Дочь A: 70% мутантных    Дочь B: 30% мутантных
                ↓                    ↓
        Деление...              Деление...
                ↓                    ↓
   Single-cell mtDNA seq → lineage tree
```

| Плюсы | Минусы |
|--------|--------|
| Неинвазивно — клетки сами делятся | Стохастический процесс — мтДНК распределяется случайно |
| Естественный баркод | Низкое разрешение: сестринские клетки могут получить одинаковую гетероплазмию |
| Работает в 3D органоидах и in vivo | Мутации накапливаются медленно — нужны поколения |
| Single-cell mtDNA seq — зрелая технология | Single-cell seq: $2-5K за эксперимент |
| Масштабируемо: тысячи клеток | Не различает сестринские клетки — только клоны через 3+ делений |
| **Nature Methods 2023:** mitochondrial lineage tracing in human haematopoiesis (Lareau et al.) | |

**Вердикт:** годится для **lineage tracing** в масштабе всей популяции. Для v3.0 — **secondary валидация**, не primary метод. Разрешение недостаточно для sister-cell судьбы.

**Ключевая статья:** Lareau CA et al. *Nat Biotechnol* 38:1197-1206 (2020). PMID: **32788668.** — mtscATAC-seq технология. Ludwig LS et al. *Cell* 176:1325-1339 (2019). PMID: **30827679.** — mtDNA lineage tracing in human haematopoiesis.

## Рекомендация: КОМБИНАЦИЯ

```
v3.0 = подход 2 (primary) + подход 3 (secondary) + подход 1 (hypothesis testing)

1. РАЗДЕЛЕНИЕ (primary): фемтосекундный лазер + манипуляторы
   → полное прогениторное дерево для N=100-200 клеток
   → судьба КАЖДОЙ дочерней клетки известна

2. мтДНК-МЕТКА (secondary): single-cell mtDNA sequencing
   → валидация lineage tree на популяционном уровне
   → независимый метод, не требующий вмешательства

3. АБЛЯЦИЯ (hypothesis testing): фемтосекундный лазер
   → "Что будет если убить клетку с mature mother centrosome?"
   → Проверка причинности ПОСЛЕ того как карта построена
```

## РЕШЕНИЕ

| Компонент | Статус |
|-----------|:---:|
| **Primary метод** | Разделение сестринских клеток (подход 2) |
| **Secondary валидация** | мтДНК-метка (подход 3) |
| **Hypothesis testing** | Абляция (подход 1) — после построения карты |
| **Ключевой вопрос** | Оптические пинцеты ($20K) ИЛИ механические манипуляторы ($3K)? |

> **Оптические пинцеты** — бесконтактно, быстро, точнее. LUMICKS C-Trap делает именно это за $200K.
> **Механические манипуляторы** — дешевле но медленнее (минуты на клетку vs секунды), риск повреждения.
>
> **РЕШЕНИЕ ПРИНЯТО (2026-07-20):** Стартуем с механических манипуляторов ($3K) для proof-of-concept. Если работает → грант на оптические пинцеты ($20K) для масштабирования.
