# README — ARGUS-OS1

ARGUS-OS1: открытая платформа для непрерывного 3D-трекинга центриолей.

**2 маркера:** Centrin1-GFP + SiR-Tubulin.
**Spindle orientation pedigreeия:** C. elegans = ∥/⟂. RPE1 = угол θ.
**Узел карты:** кластер сходных spindle orientation pedigrees (OS1) → генетическая сеть (OS3).
**Судьба:** делится/нет.

**Два предсказания:**
1. Разные spindle orientation pedigree → один узел → одинаковые потомки (конвергенция)
2. Разный возраст → один узел → одинаковые потомки (возраст НЕ влияет)

🔗 OS1 → OS2 (конвергенция) → OS3 (прогениторные карты = генетические сети).
